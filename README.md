# Good Vibe Festival

Voici le Good Vibe Festival !

Le Good Vibe Festival est un festival de musique bien connu des amateurs de musique, mais c'est aussi le meilleur endroit pour débuter votre carrière de photographe !
Réaliser des quêtes, et gagner 500  Good Vibe Point, pour devenir un véritable artiste accompli !





## A propos
Ce jeu a été réalisé en 3ème année de BUT informatique au Puy-en-Velay.

Il a été réalisé sur Unity.




## Téléchargement

Le jeu est disponible sur Itch.io : [https://nagiraud.itch.io/good-vibe-festival](https://nagiraud.itch.io/good-vibe-festival)


## Fonctionnalité:

- Visiter une ville, tout en profitant de bonnes musiques !
- Admirez vos œuvres grâce à la galerie photo !

## Contrôles:

ZQSD : Bouger 
P :  Se mettre en mode Photo
Espace : Prendre une photo (seulement quand le joueur est en mode photo !)
G :  Ouvrir votre galerie photo
E :  Ouvrir votre tableau de quêtes
